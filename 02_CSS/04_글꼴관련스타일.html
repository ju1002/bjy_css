    }

font-weight:글꼴 두께를 표현할 때 사용

    [표현법]
    선택자{
        font-weight: normal/bold/bolder/lighter/100~900
    }

font-style: 기울이고 싶을때 사용


    [표현법]
    선택자{
        font-style:normal/italic/oblique;
    }
    ->italic: 기울임 글꼴,oblique:원래 폰트를 기울임
