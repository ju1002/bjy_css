선택자(Selector)란? 중요★★★

특정 HTML요소를 선택하려고 할 때 사용하는 기능
선택자를 이용해서 원하는 요소만 쏙 선택해거 style속성을 부여

인라인 스타일 방식

1. 전체(모든요소)선택자

    [표현법]

    *{내가 부여하고 싶은 스타일 속성 : 스타일 속성 값;
    내가 부여하고 싶은 스타일 속성 : 스타일 속성 값;
    내가 부여하고 싶은 스타일 속성 : 스타일 속성 값;}
    =>현재 이 HTML문서 내의 모든 요소를 다 선택할 때 사용

2.태그 선택자

    [표현법]
    태그명{
        스타일 속성 : 값;
        스타일 속성 : 값;
        스타일 속성 : 값;
    }
    =>문서 내에 같은 태그들을 전부 선택할 때 사용

3.아이디 선택자(#아이디속성값)

    [표현법]
    #아이디 속성값{
        스타일 속성: 스타일 속성값;
        스타일 속성 : 스타일 속성값;
    }
    =>이 문서 내에서 특정 HTML요소 딱 하나만을 선택할 때 사용
    단, 해당요소에 반드시 id속성을 이용해서 이 문서내에 고유한 
    아이디값이 부여되어 있어야 함
    동일한 태그더라도 부분적을 다르게 스타일을 부여할 떄 사용
  1. 아이디 속성값
  2. 아이디 속성값
  3. 아이디 속성값
  4. 아이디 속성값

4.클래스 선택자(.클래스속성값)

    [표현법]
    .클래스 속성값{
        스타일 속성 :값;
        스타일 속성 : 값;

    }
    => 현재 HTML문서에서 여러개의 요소를 선택할 때 사용
